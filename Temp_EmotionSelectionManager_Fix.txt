РЕКОМЕНДАЦИИ ПО ИСПРАВЛЕНИЮ ОШИБОК В ФАЙЛЕ EmotionSelectionManager.cs

В методе OnConfirmEmotion нужно заменить следующие строки:

Было:
```csharp
// 2. Сохраняем запись об эмоции в Firebase
await _databaseService.AddEmotion(emotionData);

// 5. Начисляем очки пользователю
await _databaseService.AddPointsToProfile(POINTS_PER_EMOTION);
```

Должно быть:
```csharp
// 2. Сохраняем запись об эмоции в Firebase
await _databaseService.UpdateUserEmotion(emotionData);

// 5. Начисляем очки пользователю
await _databaseService.UpdateUserProfileField("totalPoints", POINTS_PER_EMOTION);
```

Метод UpdateUserEmotion определен в интерфейсе IEmotionDatabaseService, который реализуется IDatabaseService.
Метод UpdateUserProfileField определен в интерфейсе IUserProfileDatabaseService, который также реализуется IDatabaseService.

Замена этих методов исправит ошибки компиляции:
- 'IDatabaseService' does not contain a definition for 'AddEmotion'
- 'IDatabaseService' does not contain a definition for 'AddPointsToProfile'
